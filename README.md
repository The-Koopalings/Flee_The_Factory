# Flee the Factory
An educational puzzle game designed to help beginners learn fundamental programming concepts. Players are required to use their programming knowledge to navigate a robot through a room and fulfill objectives to advance to the next level. Presently, the game teaches functions, control flow, recursion, and data structures to its players.

## To Play the Game: 
1. Go into the Executables folder.
2. Download the .zip file.
3. Extract all from it. Make sure to keep all of the files/folders in the extracted folder together.
4. Open the .exe file.
